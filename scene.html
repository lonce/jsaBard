jsaBard for Storytelling with Interactive Sound Modeling

This is the jsaBard story player. To use it:

  1. Create a story using the authoring tool, which lets you load use sounds from the jsaSound library.
  2. Load the story below.
  3. Connect a device to the controller interace. For best results, try using a smartphone as a controller.
