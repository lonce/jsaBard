jsaSound for Interactive Sound Modeling

This is a subset of the sound collection and sliderbox interface described in more detail at jsaSound.

Interface experiment: Smart phone control.

Note: This can be somewhat unresponsive if you are sending control signals to the US-based animatedsoundworks.com server and back to your client. However, grab the source and run the server on a local machine, and you have a viable interface.
